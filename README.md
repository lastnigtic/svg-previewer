# svg-previewer

---

用于预览管理文件夹内`svg`文件。

### 支持功能

- 缩放文件
- 复制文件名
- 删除文件

### 配置项

```
options:
  -d, --directory <string>  relative directory to preview svg files. (default:
                            ".")
  -o, --open <boolean>      open page in browser (default: true)
  -h, --help                display help for command
```
